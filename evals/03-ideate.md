# Eval 03 — Ideate

Exercises the Ideate posture: divergent generation with critique
deliberately deferred rather than interleaved into each idea.

## Setup

Board profile at Boardroom hostility level (default). No pre-existing
session briefs required.

## Input

```text
/board give me directions for standing out in a commoditizing category
```

## Pass criteria

- [ ] Posture is detected and confirmed as **Ideate** before the board
      convenes.
- [ ] Seat takes generate directions divergently — multiple distinct
      angles, not one direction with minor variations.
- [ ] Critique of the generated directions is deferred to a clearly
      labeled section/note (e.g. "Deferred-critique notes"), not woven
      into each idea as it's introduced.
- [ ] The synthesis clusters the generated directions into named themes
      rather than listing them flat.
- [ ] The synthesis names the most promising direction(s) while explicitly
      holding that judgment loosely (this is Ideate, not Decide — no
      forced single winner).
